Anthropic Claude Code

This Week in Claude Code

Claude Code is Anthropic's command-line coding assistant — it lives right in a developer's terminal instead of a separate app, so you can hand off programming tasks without ever switching windows. This week's updates made it feel less like an experimental beta and more like a tool teams can lean on every single day.

The most useful news for anyone actually using it: Anthropic extended a promotion giving Pro, Max, Team, and eligible Enterprise users 50% higher weekly usage limits, and that bonus now runs through July 19, so heavy users get extra breathing room before hitting a cap. Under the hood, the update leaned hard into trust and safety — a new auto mode rule now blocks any attempt to tamper with session transcript files, which matters once you're letting an AI agent work somewhat independently in the background. Auto mode itself also got easier to turn on, since Bedrock, Vertex AI, and Foundry users no longer need a special opt-in flag to use it. On top of that, Claude Code picked up a real screen reader mode for plain-text rendering, small vim shortcuts like mapping "jj" to Escape, and mouse clicks that finally work inside fullscreen menus.

None of this is flashy by itself, but stacked together it shows a tool maturing past the "cool demo" phase into something built for people who need it to just work, without surprises, day after day.

Anthropic Claude Cowork

This Week in Claude Cowork

Cowork is Anthropic's mode for Claude that goes beyond chatting and actually does file and task work for you — think of it less like texting a smart friend and more like handing a capable assistant the keys to your desktop. This week Cowork officially broke out of its cage. It started life as a desktop-only feature, but Anthropic just announced it's spreading to the web and to phones, with the rollout beginning July 7 for Claude Max subscribers before it trickles down to other plans.

The headline change is that Cowork tasks no longer need your laptop to stay open and awake. You can kick off a task on your desktop, close the lid, and Claude keeps working in the background — even scheduled tasks can run with zero personal devices online. Then you can check progress or redirect the work from your phone later, like leaving a contractor working on your house and just checking in through a doorbell camera instead of standing over their shoulder all day. Anthropic also merged Chat and Cowork into a single home tab, so there's now one sidebar, one search bar, and one shared home for Projects and Artifacts instead of two separate worlds. On top of that, Claude picked up write access to Microsoft 365 tools, meaning it can now draft and send emails, manage calendar events, and create or update OneDrive and SharePoint files directly.

To celebrate the expansion, Anthropic is keeping its doubled Cowork usage limits in place through August 5. The bigger picture here is that Anthropic is betting Cowork becomes less like a chat window and more like an always-on coworker who never needs you to babysit it — a meaningful shift for anyone using Claude to manage real work instead of just answering questions.

🔍 Tool Spotlight

ZML / LLMD

ZML is a Paris-based AI infrastructure startup, and this week it launched LLMD, a free server that lets AI models run efficiently across almost any chip — Nvidia, AMD, Google's TPUs, Apple's Metal chips, and Intel Arc included. Normally, getting an AI model to run fast means being locked into whichever chip it was built for, kind of like a video game that only works on one console. LLMD acts as a universal adapter, letting popular open-source models like Llama, Mistral, Gemma, and Qwen run at full speed no matter what hardware is underneath, with an interface that's compatible with the same API standard OpenAI uses.

Why it matters: AI companies are currently fighting over scarce, expensive GPU capacity, and LLMD gives developers more freedom to shop around for cheaper or more available chips instead of being stuck. It's aimed at developers and companies running their own AI models rather than everyday users, but its ripple effects — cheaper, faster AI services — could reach everyone. What's surprising is the backing: ZML counts Turing Award winner Yann LeCun, and the founders of Hugging Face and Docker among its investors, a serious vote of confidence for a 20-person startup barely a year old.

🔍 Tool Spotlight

Cursor

Cursor is a code editor built around AI, essentially a supercharged version of the familiar VS Code that programmers have used for years, except now you can describe what you want in plain English and an AI agent will actually write, edit, and run the code across your project's real files. It's become one of the most popular tools among professional developers precisely because it treats AI not as an autocomplete gimmick but as a coworker that can take on entire chunks of a task.

This week's version, 3.11, released July 10, adds something called side chats: little parallel conversations you can spin off from your main AI session using a quick "/side" command, without derailing whatever the main agent is busy building. You might ask a side chat to research an alternative approach or sanity-check a decision, then pull that answer back into the main thread later, or just leave it running like a separate open tab. The update also makes it easier to search back through old agent conversations and simplifies picking which project or repository you're working in.

What's genuinely interesting here is what it says about where AI coding tools are headed: instead of one AI doing one thing at a time, Cursor is now managing multiple simultaneous AI conversations the way a person juggles browser tabs. It's a small feature, but it hints at coding agents evolving from single-threaded assistants into something closer to a small team you're directing at once. Anyone who writes code for a living, from students to professional engineers, is a plausible user here.
